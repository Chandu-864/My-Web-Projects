PROJECTS

This Weather App allows users to check the current weather of any city by fetching real-time data from the OpenWeather API. It displays the temperature, humidity, wind speed, and updates weather icons based on the conditions (e.g., clouds, rain, clear sky). Users can input a city name and get results by clicking the search button or pressing the "Enter" key. The app also handles invalid inputs, showing an error message for unrecognized cities. Built with HTML, CSS, and JavaScript, the app demonstrates my skills in API integration and responsive design.
